# Giants from Dust

Build modern AI models from the ground up — no magic, no hand-waving. Understand exactly what happens when a neural network trains, how attention works, and how GPT-style models are constructed.

## Notebooks

| Notebook | Description |
|----------|-------------|
| [`notebooks/NN_from_scratch_v1.ipynb`](notebooks/NN_from_scratch_v1.ipynb) | Neural networks from scratch with NumPy — XOR, forward pass, backprop, and PyTorch comparison |
| [`notebooks/text_to_attention.ipynb`](notebooks/text_to_attention.ipynb) | From text to attention — tokenization, embeddings, positional encoding, and attention coded from scratch |
| [`text_to_attention_2.ipynb`](text_to_attention_2.ipynb) | Continued exploration of text and attention |
| [`text_to_attention_withembeddding.ipynb`](text_to_attention_withembeddding.ipynb) | Text-to-attention with embeddings |
| [`notebooks/01_gpt_from_scratch`](notebooks/01_gpt_from_scratch) | GPT implementation from scratch |
| [`AQ_GPT_V1.ipynb`](AQ_GPT_V1.ipynb) | GPT variant implementation |
| [`modernGpt.ipynb`](modernGpt.ipynb) | Modern GPT architecture |
| [`PlayingWithAgents.ipynb`](PlayingWithAgents.ipynb) | Experiments with AI agents |

## Getting Started

1. Clone the repository
2. Open any notebook in Jupyter or [Google Colab](https://colab.research.google.com) (most notebooks include Colab badges)
3. Run the cells in order — each builds on the previous

## Requirements

- Python 3
- Jupyter
- NumPy
- PyTorch (for some notebooks)
- tiktoken (for text/tokenization notebooks)
